I have some MP3's which can't be found. What can I do?

With some MP3's you will get a Found: Not found-"Album Title" if the Album-Artist tag isn't set properly in the MP3 itself.

LibraryTag only shows "This is a false MBID", nothing more.

Install openJDK1.8.0_461! The main program would work with e.g Java-25, but the underlying MusicBrainz-Client doesn't. Later, when the program is fully implemented, I will search for a solution, but for now Java 8 is the only choice.

On Windows I have Java-25 installed and want to execute the program under a secondary JDK. How can I accomplish this?

Install openJDK1.8.0_461! Then create a startup.cmd file in the program directory with these lines:

setlocal
set JAVA_HOME=Path\to\JDK\JDK1.8.0_461
set JRE_HOME=%JAVA_HOME%\JRE
set PATH=.\;%JRE_HOME%\bin;%JAVA_HOME%\bin;%PATH%
%JRE_HOME%\bin\javaw.exe -jar "Path\to\Jar\LibraryTag.jar"
endlocal.

Create a link on your desktop which starts startup.cmd.
