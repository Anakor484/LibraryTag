0.03 Pre-View
• Changed the names in the Properties file. Delete LibraryTag.prp if available.
• Added 4 new properties:
    → MB.Limit – sets the limit of lookup requests (default 100)
    → LibraryTag.Update [true/false] – main switch for accessing the library
    → LibraryTag.Basedir – position of the library (Win example: "c:\", Linux/Unix default "/")
    → LibraryTag.RW [ro/rw] – readonly or readwrite access
• Added searching for files (currently readonly)
• Readme updated to mention LibraryTag.prp

• Removed FlatLaf support (because of native library issues)
