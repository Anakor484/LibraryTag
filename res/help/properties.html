LibraryTag.prp is an automatically created plain text file. It resides in the program's home directory. The lines are key/value-paired settings.
 

A typical file contains these lines:

#LibraryTag - Properties File
#Sat Nov 15 07:21:18 CET 2025
LibraryTag.Basedir=m:\\
LibraryTag.Height=626
LibraryTag.LF=Metal
LibraryTag.RW=ro
LibraryTag.Update=true
LibraryTag.Width=908
LibraryTag.X=32
LibraryTag.Y=141
MB.Limit=100

Comments:
Lines beginning with a '#' are comments. The first line is the program name; the second line is the date and time of creation.

LibraryTag.X, LibraryTag.Y:
These are the last window coordinates. If the program fails to start due to invalid coordinates, reset them to 0.

LibraryTag.Width, LibraryTag.Height:
The width and height of the program's main window.

LibraryTag.LF:
This setting defines the Java Look & Feel, which determines the appearance of the app. Valid values are: Metal, Nimbus, GTK, CDE/Motif, Windows. If this line does not exist, a new line will be created based on the operating system. If the Line includes a non existent Look & Feel the default Metal will be loaded.

LibraryTag.Basedir:
This sets the drive, mount point, and path of your music library. On Windows, m:\\ represents the escaped drive "m:\". On Linux, a path like /mnt/m/ would be valid.

LibraryTag.Update:
If set to true, access to the Media Library is enabled. If set to false, no access is allowed.

LibraryTag.RW:
This setting can be ro (read-only) or rw (read-write). Tags will be written to files only if rw is set.

MB.Limit:
This sets the limit for a single MusicBrainz query.
