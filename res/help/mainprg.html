The main window currently has 5 elements.
A text field for input, a text area for output. If you paste a valid MBID the program will fetch the corresponding data from MusicBrainz. The Go button on the right submits the query. The Clear button below clears the output area.
MBIDs appearing in the output area can be selected with the mouse, copied to the clipboard (Ctrl+C) and pasted directly into the input field (Ctrl+V) to perform another query.

Keyboard shortcuts:

KeyAction
F1 Opens this help
F5 Copy the last used MBID back into the input field
F8 Clear the input field
F9 Activate Menu
ENTER Submit the query
